# Retail Sales Analytics Dashboard

## Project Overview

This project presents an interactive Retail Sales Analytics Dashboard built using Tableau to analyze sales performance, customer behavior, product performance, and geographical sales trends.

The dashboard enables business users to monitor key performance indicators (KPIs), identify high-performing products and customers, and make data-driven decisions through interactive visualizations.

---

## Dashboard Features

- Executive Dashboard
- Country & Product Insights Dashboard
- Customer Insights Dashboard
- KPI Cards
- Monthly Sales Trend
- Sales by Category
- Sales by Country
- Top 10 Products by Sales
- Top 10 Customers by Sales
- Interactive Filters
- Navigation Buttons

## Key Performance Indicators (KPIs)

- Total Sales
- Total Orders
- Total Customers
- Average Order Value
- Sales per Customer
- Total Quantity Sold
- Total Products
